# Qué

El Hacker Club es un vehículo de networking e intercambio de experiencias centrado en emprendedores o empresas con un proyecto cuyo foco sea digital o muy tecnológico.

# Quién

El Hacker Club sólo está abierto a empresas o emprendedores con proyecto propio, excluyendo a empresas de venta de servicios, i.e. consultoras - esto es así para evitar conflictos de interés y tener foco en los problemas que tienen las startups: financiación, herramientas, etc.

# ¿Quieres entrar?

Envia un mensaje a antorsae+NUMERO@gmail.com donde NUMERO es el número primo número 100,000 (el cienmilésimo número primo, vamos) explicando tu proyecto (en el Hacker Club tenemos desde empresas consolidadas a proyectos que están empezando).
En cualquier caso querremos conocerte antes.
